제 공부 정리하는 김에 노트 정리가 힘드신 분들을 위해 공유합니다. 목차 링크 클릭시 해당 섹션으로 이동합니다.

텍스트의 경우 직접 작성해 공유나 수정하셔도 되지만 이미지 등의 리소스, 예제 코드의 경우 MDN, Geeks4Geeks, W3CS 등에서 가져온 것으로 학습용으로 활용해 주시기 바랍니다.

MarkDown(`.md`) 파일에 내용이 있습니다.

## 핸드폰 및 윈도우에서 MarkDown 보기

계속 수정이 되므로 웹으로 보시는 걸 추천드리지만 GitHub가 아니라 저장 후 읽고 싶으시다면 모바일, 윈도우 환경에서는 Obsidian 이용 추천드립니다. Obsidian 폴더 설정후 그 곳에 zip 다운 받은걸 푸시면 됩니다.

Linux의 경우 VSCodium/VSCode에서 `Ctrl+Shift+V` 누르시면 preview 뜹니다! 별도 프로그램 설치하실 경우 Zettlr도 추천드립니다.
